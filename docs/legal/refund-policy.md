# ENHE Promotion Manager Refund Policy

Effective date: 2026-07-10

This is a launch draft. Review with counsel and align it with Stripe settings before public launch.

## Subscription Refunds

For paid monthly subscriptions, users may request a refund within 7 days of the first payment if no hosted run credits have been consumed.

## Credit Usage

Credits consumed by completed hosted runs are generally non-refundable because server resources have already been used. If a hosted run fails before execution, unused reserved credits should be returned automatically by the backend.

## Duplicate Or Accidental Payments

Duplicate charges or accidental plan upgrades may be refunded after verification.

## Platform Review Outcomes

Chrome Web Store, Microsoft Edge Add-ons, YouTube, Zhihu, Xiaohongshu, Douyin, GitHub, and other third-party review decisions are outside ENHE control and are not guaranteed.

## Request Support

Submit refund or billing questions at:

https://www.enhe-tech.com.cn/promotion-manager/support

Include license email, Stripe receipt ID if available, plan name, and a short description of the issue.
