# ENHE Product Promo Maker Support

Support for ENHE Product Promo Maker (formerly ENHE Promotion Manager) is available through the public support URL below.

## Support URL

https://www.enhe-tech.com.cn/promotion-manager/support

## Support Scope

- Extension installation and store package questions.
- License validation and billing portal access.
- Hosted-run queue status.
- Credit reservation and failed-run refunds.
- Publish package generation issues.
- Security or privacy questions.

## Information To Include

- Browser and extension version.
- License plan, but never paste payment card data.
- Hosted run ID, if available.
- Product URL and target platform list, if relevant.
- Screenshot or error text.

## Response Target

For launch, target 2 business days for normal support and 1 business day for billing/security issues.
